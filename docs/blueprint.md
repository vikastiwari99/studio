# **App Name**: MathMentorAI

## Core Features:

- Profile Customization: Users select their grade level, preferred topics, and difficulty levels.
- Problem Generation: Generates math problems based on user-selected parameters. Uses AI to tailor problems to the selected grade, topic and difficulty.
- Hint System: Provides step-by-step hints or partial solutions to aid students when they are stuck. Uses AI reasoning as a tool to help construct hints based on common problem-solving techniques and user profile
- Complete Solution: Displays a full, detailed solution for each problem to help students understand the correct approach.
- User Support: Connects students with tutors via chat. (future)
- Progress tracking: Monitor students' progress via database to allow future tailored generation of questions (future)

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) for trustworthiness and focus.
- Background color: Light blue (#E5F3FF), a very pale shade of the primary, creating a calm learning environment.
- Accent color: A calming green (#33FF99) for positive reinforcement.
- Body and headline font: 'Inter', a sans-serif for readability and a clean user experience.
- Simple, intuitive icons representing math topics (e.g., division symbol, graph icon).
- Clean, uncluttered layout to minimize distractions and maximize focus on problem-solving.
- Subtle, non-distracting animations to provide feedback during problem generation and solution display.
